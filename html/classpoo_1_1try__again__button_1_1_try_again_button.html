Detailed Description

Clase que representa un botón de "Reintentar" al finalizar el juego. Esta clase muestra una pantalla con el mensaje "Fin del juego", la puntuación actual del jugador y las mejores puntuaciones. Además, permite al jugador reintentar el juego con el botón "Reintentar".
