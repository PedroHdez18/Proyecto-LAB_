\doxysubsection{Detailed Description}
Clase encargada de manejar el bucle principal del juego, incluyendo el movimiento de la serpiente y la detección de colisiones. 

Esta clase también interpreta las entradas del teclado para dirigir a la serpiente y gestiona las interacciones con elementos del escenario, como la comida.
